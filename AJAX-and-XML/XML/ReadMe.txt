- XML is eXtensible Markup Language
- Markup Language for documents containing structured information
- We don't use predefined tags like HTML in XML 
- We use custom tags and custom attributes
- Somewhat like database to store information 
- XML is self-descriptive language
- XML is both human and machine readable 
- We can store text, image(binary data)

- Components of XML is like HTML: 
	- Custom Root Tag
    - Element
    - Attributes
    - Entities: &It;(<)
    - CData

- Rules of XML:
    - XML declaration is the first statement
        - Eg: <?xml version = "1.0" encoding = "UTF-8"?>
    - Every document must contain a root element

- Element Types in XML: 
    1. Simple Element
    2. Complex Element

- Empty Element Example: 
    - <name id="123"/>

- XSD means XML Schema Definition 

- Download XML viewer form: https://www.mindfusion.eu/xml-viewer.html and Install usign "setup" file
